## Pembelajaran repository ini mencakup konsep dasar seperti:
*   1  . Struktur Dasar Program Go
*   2  . Variabel dan Konstanta (string, int, float64, bool, byte, rune, complex64)
*   3  . Input dan Output (fmt package)
*   4  . Control Alur (if-else, switch)
*   5  . loops (for loop, while style, forever loop)
*   6  . Array dan Slice (array, slice)
*   7  . Map (dictionary) {penjelasan dan contoh}
*   8  . Fungsi (fungsi deklarasi, multiple return values)
*   9  . Struct (struct deklarasi, dan penggunaan)
*   10  . Methods (method deklarasi, dan penggunaan)
*   11  . Pointer (pointer deklarasi, dereferencing, dan penggunaan)
*   12  . Goroutine dan Channel (konsep dasar concurrency di Go)
*   13  . Channels (deklarasi, pengiriman, penerimaan data)
*   14  . Error Handling (penanganan error dengan multiple return values)
*   15  .  File I/O (membaca dan menulis file)
*   16  . Package dan Modul (membuat dan menggunakan package, modul go)
*   17  . HTTP Server (membuat server web sederhana dengan net/http package)
*   18  . Json Handling (encoding/json package untuk parsing JSON)
*   19  . Testing (testing package untuk unit testing)
*   20  . Deployment (cara build dan deploy aplikasi Go)

## Setiap topik akan dijelaskan dengan contoh kode yang sederhana dan mudah dipahami.
